## Stack

Static HTML site. No build step.

- HTML5 + CSS3
- [HTML5 UP – Read Only](https://html5up.net/read-only) template (CCA 3.0 license)
- Vanilla JS (jQuery via local assets)

## Run locally

```bash
# Any static server works, e.g.:
npx serve .
# or
python3 -m http.server
```

Then open `http://localhost:3000` (or whatever port the server prints).
